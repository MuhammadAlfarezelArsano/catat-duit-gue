# 💰 Catat Duit Gue Bot (Telegram)

Bot asisten keuangan pribadi sederhana yang bisa mencatat pengeluaran/pemasukan lewat Telegram dengan gaya bahasa santai.

## ✨ Fitur
- **Natural Language Parsing**: Bisa baca "15rb", "50k", atau "2jt".
- **Real-time Recording**: Langsung memberikan konfirmasi setelah input.

## 🛠️ Tech Stack
- **Node.js**
- **Telegraf.js** (Telegram Bot Framework)
- **Dotenv** (Environment Variables)

## 🚀 Cara Menjalankan
1. Clone repo ini.
2. Jalankan `npm install`.
3. Buat file `.env` dan masukkan `BOT_TOKEN` lo.
4. Jalankan `node index.js`.
